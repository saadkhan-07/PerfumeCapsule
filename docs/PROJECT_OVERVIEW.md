# Project Overview

> **Status:** Placeholder — to be expanded in a later phase.

Perfume Capsules is a modern, mobile-first ecommerce platform for selling authentic
branded perfume **decants** (5ml, 10ml, 30ml, etc.) rather than full-size bottles.

The business operates primarily through **WhatsApp ordering**. The website showcases
products, manages users and carts, collects shipping information, persists orders, and
then redirects the customer to WhatsApp with a pre-filled order message. Payments are
handled manually via **JazzCash** or **EasyPaisa** after the WhatsApp conversation begins.

See [`../CLAUDE.md`](../CLAUDE.md) for the authoritative project description, goals,
tech stack, and build phases.
