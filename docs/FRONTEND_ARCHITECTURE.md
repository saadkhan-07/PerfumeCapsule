# Frontend Architecture

> **Status:** Placeholder — to be expanded in Phase 4 (Frontend Foundation).

The frontend is a React + Vite + TypeScript + Tailwind CSS application. It is
component-driven: reusable UI lives in `components/ui`, business-specific UI lives in
`features`, and presentation components stay free of business logic.

Refer to the **Frontend Structure** section of [`../CLAUDE.md`](../CLAUDE.md) for the
folder layout until this document is completed.
